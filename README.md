# 🎨 Material Debugger Tool for Blender

**Material Debugger Tool** 是一款专为 Blender 4.2+ (EEVEE NEXT) 开发的材质高级调试插件。

它允许你在 3D 视图中以“所见即所得”的方式，实时且精准地检查复杂材质节点网络中每一个像素背后的真实数学数值。无论是大于 1 的过曝高光，还是隐藏在暗部的负数数据，它都能将其准确地可视化并打印在屏幕上。彻底告别在视口中“凭感觉瞎猜”，让材质逻辑的构建变得绝对严谨。

### **注意:** 插件使用了实时合成器制作,因此仅支持在EEVEE NEXT中运行。

---

## ✨ 核心特性

* 🚀 **一键无痕配置**：点击开启调试，插件会自动在后台配置渲染层、AOV 通道和合成器节点树。关闭调试时，自动恢复你原有的合成器连接。

* 🖱️ **快速启动调试**：使用 `Shift + Alt + 左键` 点击任意材质节点，即可自动将其数据输出至屏幕。
* 🎯 **实时指针追踪**：开启指针模式后，准星将跟随鼠标在 3D 视图中实时移动，并在准星旁动态显示鼠标悬停处像素的精确数值。

---

## 🛠️ 安装环境与要求

* **Blender 版本**：`4.2.0` 或更高版本。
* **渲染引擎**：必须使用 **EEVEE NEXT**（由于依赖实时的屏幕空间 AOV 与视图合成特性，暂不支持 Cycles 或旧版 EEVEE）。

---

## 📦 安装说明

1. 在 Release 页面下载最新的 `.zip` 文件。**请不要解压它！**
2. 打开 Blender，前往顶部菜单栏：`编辑 (Edit)` -> `偏好设置 (Preferences)`。
3. 在左侧选择 `插件 (Add-ons)` 选项卡。
4. 点击右上角（或右上角下拉菜单）的 **安装 (Install)** 按钮。
5. 在文件浏览器中找到你刚刚下载的 `.zip` 文件并双击。
6. 在插件列表中找到 **Generic: Material Debugger Tool**，**勾选其前面的复选框**以启用插件。

---

## 🚀 快速入门

### 1. 开启调试模式
打开 **着色器编辑器**，使用 `Shift + Alt + 左键` 点击任意材质节点开启预览(**非Shader类型**),在N面板对应选项卡中可以调整显示参数,点击关闭调试会回复开启前的合成器状态。

### 2. 使用指针采样像素
在预览模式开启后，点击 **指针模式** 按钮。
将鼠标移动到 3D 视图中，准星会跟随鼠标移动，并在屏幕上实时显示出当前像素的数值。再次点击该按钮关闭。

### 4. 退出调试
点击N面板中的**关闭调试** 按钮关闭调试模式,也可以手动在合成器中删除对应节点关闭调试。

---

## ⚙️ 界面参数详解

* **指针模式**: 仅显示鼠标位置的数值。
* **显示模式**:
  * `Auto`: 自动模式。
  * `Gray`: 强制将数据映射为灰度值显示。
  * `RGB`: 强制显示 RGB 通道色彩。
  * `HSV`: 将数据映射为色相、饱和度、明度显示。
* **显示边框**: 显示每一个分块的边框。
* **显示底色**: 仅显示数值不对颜色进行分块。
* **缩放与偏移**:
  * 包含 `数字X/Y缩放`、`文字X/Y偏移`、`指针大小` 等选项，用于调节 HUD (屏幕显示界面) 元素的大小和位置。
